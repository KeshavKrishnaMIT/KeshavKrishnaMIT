**React 19 • Express.js • Supabase • JavaScript • Vercel**

A production-style CRM platform built for study-abroad consultancies, consolidating student lifecycle management, counselor workflows, analytics, and AI-assisted decision support into a unified dashboard.

**Highlights**
- Intelligent university recommendation engine using weighted academic, financial and destination scoring
- Student Lead Management System with complete admission pipeline
- Analytics dashboard with real-time business KPIs and conversion metrics
- Counselor assignment, follow-up tracking and workflow automation
- AI-assisted operational insights generated from live CRM datasets
- Student segmentation across 5,000+ records with interactive filtering
- RESTful Express backend integrated with Supabase
- Responsive dashboard built using reusable React components
